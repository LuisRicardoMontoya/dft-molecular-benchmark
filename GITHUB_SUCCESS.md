## 🚀 **CÓMO USAR EL REPOSITORIO**

### Clonar:
```bash
git clone https://github.com/LuisRicardoMontoya/dft-molecular-benchmark.git
cd dft-molecular-benchmark
```

### Instalar dependencias:
```bash
python3 -m venv dft_env
source dft_env/bin/activate
pip install ase gpaw matplotlib numpy scipy pandas rdkit
```

### Ejecutar pipeline:
```bash
python3 run_all.py
```

### Ver resultados:
```bash
cat results/molecular_summary.csv
open pdf/DFT_Molecular_Benchmark_Professional.pdf
```
